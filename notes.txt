   Consigne Voiture : 
C’est un site non visuel (pas de template à reproduire) mais faire travailler votre aspect créatif et avoir un style et un design aussi poussé que les autres site proposer
Avoir une page home avec
Une navbar pour pouvoir naviguer, il y aura sur la nav (bouton pour naviguer entre les pages, icon pour se connecter, icon pour aller dans notre page panier)
En header avec un carrousel avec 5 voitures
Une section avec des voitures des années 90 (90 -> 99)
Une section avec des voitures qui coûte moins de 20 000$.
Quand on clique sur une des voitures, ça nous dirige vers sa page details (route id)
Chaque view des voitures à sa propre google maps + chaque détails de celle-ci.
Voiture achetée. Celle-ci disparaît.
Bonus : Contacter le vendeur. Pour pouvoir acheter, le vendeur doit se connecter et lui envoyer un message (créer une messagerie). il pourra se connecter via un autre compte pour pouvoir y répondre
Bonus 2 : Filtre sur une map.

Le site doit être fait en NextJs (c’est une évidence ;) )
Vous allez travailler avec une API pour les trois propositions
Logo de chargement lorsque ça fetch api
Bar de recherche/ filtre
vue détaillée => view sur l' article = routes uniques.
multi-page
carrousel.
Form/ connexion :
un système de panier (supprimer/ajouter/modifier) avec la condition d'être connecté.
Animation quand on arrive sur un component (loading.. fading).
Et bien sûr qui dit front, dit responsive ;)
